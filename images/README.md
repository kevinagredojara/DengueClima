# Imágenes

Esta carpeta contiene las visualizaciones y gráficos generados por el proyecto.  

## Contenido

| Archivo | Descripción |
|---------|-------------|
| `dengue_vs_precipitacion_cali_2020.png` | Gráfico comparativo de casos de dengue y precipitación en Cali (2020) |
| `casos_por_año. png` | Distribución histórica de casos de dengue por año |
| `mapa_calor_municipios.png` | Mapa de calor de incidencia por municipio |

## Uso

Estas imágenes se utilizan en: 
- Documentación del proyecto
- Notebooks de análisis
- Presentaciones

## Nota

Las imágenes se generan automáticamente desde los notebooks.  
Para regenerarlas, ejecuta los notebooks correspondientes.
